# Tuna - 庄周吃鱼

A modern open source AI-powered application platform.

庄周吃鱼：万事不决问庄周。

利用 AI、大数据、云计算等技术解决"X 条件下如何选择 Y"的问题。

## 项目理念

Tuna 是一个现代化的 AI 驱动应用平台，用户可以创建和运行自己的应用。我们不断增加应用类型和模板，提供从简单决策到复杂数据驱动的多样化解决方案。

## 核心特性

- 🚀 **应用构建平台** - 用户无需代码，创建和部署自己的智能应用
- 🤖 **AI 赋能** - 集成 AI 能力，提供智能决策支持和数据分析
- 🌍 **多语言支持** - 原生国际化，支持全球用户
- 🎨 **现代设计** - 极简主义风格，优雅的用户交互体验
- 🔐 **安全可靠** - 企业级认证和数据保护机制

## 应用示例

用户可通过 Tuna 创建以下类型的应用：

- **决策工具** - 中午吃什么、周末去哪玩等日常选择辅助
- **抽签/抽奖** - 年会抽奖、班级点名、多人互动决策
- **数据驱动应用** - 宝宝餐单推荐（基于时间和收集数据）、智能计数器
- **二选一工具** - "生存还是毁灭"式的快速决策
- **自定义应用** - 用户可灵活构建任何基于选择和数据的应用

## 技术栈

- **前端** - Next.js 15 + React 19 + TypeScript
- **样式** - Tailwind CSS + shadcn/ui
- **后端** - Supabase (PostgreSQL + 认证）
- **部署** - Cloudflare Pages

## 本地开发

```bash
npm install
npm run dev
```

访问 `http://localhost:3000` 开始使用。
